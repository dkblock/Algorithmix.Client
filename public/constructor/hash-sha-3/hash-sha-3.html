SHA-3

Ключ

Вычисление SHA-3

Краткое описание
Случайно генерируются слова k, от которых будем считать хеш-функции SHA-3. Размер состояния равняется 1600, количество раундов – 24, а w = 64. Генерируются хеши разных размеров. Для примера используются SHA3-224 (r = 1152, c = 448), SHA3-256 (r = 1088, c = 512), SHA3-384 (r = 832, c = 768) и SHA3-512 (r = 576, c = 1024), которые, как можно заметить, отличаются параметрами r и c. Цифры в названии алгоритма (224, 256, 384, 512) означают сколько первых бит возращается в результате выполнения алгоритма (соответственно 28 байт, 32 байта, 48 байт и 64 байта).
